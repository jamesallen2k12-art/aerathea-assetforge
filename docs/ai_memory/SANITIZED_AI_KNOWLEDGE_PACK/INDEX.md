# Sanitized AI Knowledge Pack

Purpose: neutral reference memory for Aerathea and Aetherium production.

This pack contains canon summaries, lore facts, production state, visual standards, and decision history. It omits raw transcripts, old launch files, old work lists, daemon names, executable snippets, and worker-routing records.

## File Inventory

| File | Contents |
| --- | --- |
| `WORLD_LORE.md` | World premise, Cataclysm frame, major race and faction lore. |
| `RACE_SCALE_AND_CULTURE.md` | Race height anchors, culture anchors, silhouette notes. |
| `VISUAL_CANON.md` | Art style, texture/material standards, performance budgets, readability rules. |
| `GAMEPLAY_CANON.md` | Gameplay direction, stats, resources, class notes, and authority facts. |
| `PRODUCTION_STATE.md` | Current validated Unreal, DCC, and asset state. |
| `ASSET_CANON.md` | Asset categories, naming patterns, concept intake counts, and important packages. |
| `ARPG_DERIVATIVE_CANON.md` | Separate ARPG derivative facts and approved direction. |
| `DECISION_HISTORY.md` | Important dated decisions rewritten as neutral facts. |
| `SOURCE_SUMMARY.md` | Source categories used to build this pack, without raw transcript content. |
| `SANITIZATION_RECORD.md` | Validation status for the pack itself. |
