## Use For A1

Use this reference set to improve implementation details while preserving the approved A1 visual target at `docs/assets/visual_canon/BloodAxeCairnTargets_A01/VC_GIA_BloodAxe_CairnTarget_A1.png`.

Useful lessons:

- Stone should read as broken, faceted, dark cairn rock with brighter chipped planes.
- Red paint should read as worn pigment integrated into the slab surface, not raised red bars.
- Rope/rawhide should be rounded, wrapped, and knotted rather than rectangular straps.
- Ground contact needs more rubble and mud scatter around the front and side silhouette.
- Side/back stones need authored 360-degree massing, not a flat front-only approximation.

Do not use the larger shrine, throne, camp, or stacked-cairn concepts to redefine A1. They are secondary material/detail references only.
