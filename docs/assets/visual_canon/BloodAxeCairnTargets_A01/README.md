## Required Workflow

1. Build one target at a time.
2. Keep the exact target image open during DCC work.
3. Render DCC proof beside the target image before Unreal import.
4. Match major silhouette, stone count/massing, cloth/rawhide placement, red paint placement, base shape, and side/back information.
5. Do not use round generic bases unless the target image clearly calls for that shape.
6. Do not use one target as a generic style reference for another target.
7. Do not promote any cairn to final visual approval until its concept-target comparison is approved.

## Boundary

This folder stores visual targets only. It does not approve gameplay behavior, route logic, waypoint use, quest marking, VFX/audio, destructible behavior, collision finalization, or `Fully game-ready` status.
