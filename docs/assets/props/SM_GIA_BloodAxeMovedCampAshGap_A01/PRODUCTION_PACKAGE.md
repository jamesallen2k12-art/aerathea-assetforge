It should read as a cold ash and trampled mud interruption between visible cairn remnants, not as a trail decal, path instruction, damage field, aura field, gatherable ash, VFX source, audio source, route validator, or gameplay marker.

All future static mesh implementation requires LOD0, LOD1, LOD2, and LOD3 before shipping use.

- LOD0: full irregular footprint, shallow relief, optional embedded stones, and optional low cloth fleck.
- LOD1: reduce edge subdivisions, flatten minor relief, and simplify embedded stones.
- LOD2: preserve only the broad ash/mud footprint and largest value breaks.
- LOD3: flattened low proxy or material-only representation if later implementation permits it.

Reduction order:

1. Ash flecks, soot speckles, mud streaks, burned grass fibers, tiny cracks, and minor pitting.
2. Small edge wobble, tiny stones, cloth fray, and optional residue details.
3. Secondary embedded stones and non-silhouette relief.
4. Broad footprint shape only after all small detail is removed.

Never preserve micro-detail at the expense of the broad non-directional ash gap read.
