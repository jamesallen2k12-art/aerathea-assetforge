## Required Audit Gates

The A002 handoff audit must prove:

- A21 source evidence is named as the upstream authority.
- No quarantined A002 Phase 3C through Phase 7D output is read or copied.
- A002 package geometry bounds match A21 within declared tolerance.
- A002 package LOD triangle counts match A21: `[13300, 4628, 1812, 716]`.
- Visible texture pixels remain exact.
- Atlas source regions remain exact.
- Contact gaps remain zero.
- Center offsets remain zero.
- Rotations remain zeroed.
- Meshes remain closed with no boundary or non-manifold edges.
- Collision proxy count remains `2`.
- Any changed names/paths are listed in a rename map.

The gate must fail if:

- any quarantined A002 proof-shell asset is referenced
- any visible texture is filtered, resized, or recolored
- any geometry is regenerated by analytic oval/proof-shell logic
- any A21 geometry, UV, collision, or material slot content changes without a written approved reason
- any import/capture work starts before the handoff gate passes
