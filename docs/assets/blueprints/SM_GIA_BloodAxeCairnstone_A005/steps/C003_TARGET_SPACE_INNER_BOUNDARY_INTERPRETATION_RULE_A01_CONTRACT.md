perimeter in a construction-origin centered target frame and conditionally
approves a four-sector, order-preserving CL-003 mapping convention. That
mapping cannot execute because no approved C-003 target-space boundary exists.

coordinate, map any CL-003 sample, implement `S10R-003-A` or `S10R-006-A`,
revise Step 10, or create geometry.
