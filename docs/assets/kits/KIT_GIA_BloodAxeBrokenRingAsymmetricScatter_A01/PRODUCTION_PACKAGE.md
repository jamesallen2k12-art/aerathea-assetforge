`KIT_GIA_BloodAxeBrokenRingAsymmetricScatter_A01` defines an asymmetric broken-ring scatter with irregular spacing, disturbed sockets, and low ash/mud ground contact. The intended read is a ruined Blood Axe camp perimeter memory: former ring stones have shifted, fallen, been dragged away, or settled unevenly into old ash and mud, leaving an interrupted scatter that suggests history without creating a usable route or gameplay signal.

This asset must read as visual memory only, not route language. The scatter may imply that a hostile Giant raider camp once marked the site, but it must not read as a waypoint, quest marker, objective marker, route marker, route plan, navigation/pathfinding cue, arena edge, encounter layout, active ritual boundary, or implementation target.

Neutral/civilized Giant culture remains associated with hidden cave towns, master stonework, terraces, bridges, waterworks, hearth light, blue-gray civic masonry, and restrained storm or rune accents; none of those are the default read for this package.
