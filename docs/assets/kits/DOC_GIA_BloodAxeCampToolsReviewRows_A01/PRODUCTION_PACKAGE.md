## 3. Silhouette Notes

Review rows should compare composition density across buckets, coils, hooks, wedges, mallets, tie hardware, and clusters. The intended visual standard is chunky, separated forms with readable negative space.

Policy target: primary forms should remain readable at MMO camera distance before texture detail is considered. Reject rows where tiny props dominate, where clutter becomes a wall, or where cluster silhouettes read as treasure piles, UI markers, route signs, or human-scale workshop clutter.
