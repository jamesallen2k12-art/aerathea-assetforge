# KIT_ABY_ShadowFlame_A01 Source Comparison

## Purpose

This file compares the 10 generated Abyss lore images in `docs/lore/images/abyss-shadow-flame/` against the live source concepts in `/home/Flamestrike/Desktop/Aerathea/Aerathea Creative/ASSET CONCEPTS/`.

## Result

The generated lore set is useful as a named elite/boss exploration set. The live source concepts are stronger for production hierarchy because they add troops, formations, hounds, shield units, ranged units, and Anathema siege-drake variants. The kit should use both:

- Generated lore images: narrative names, elite roles, boss mood, and magic themes.
- Live source concepts: practical roster breakdown, silhouettes, scale tiers, and first child package candidates.
