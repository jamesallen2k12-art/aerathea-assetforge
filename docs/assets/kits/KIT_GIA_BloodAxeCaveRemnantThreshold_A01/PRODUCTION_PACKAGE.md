`KIT_GIA_BloodAxeCaveRemnantThreshold_A01` is a small static threshold composition for abandoned Blood Axe cave-edge memory dressing. It groups low cairn mass, fractured low stone, restrained old cloth, ash/mud base, and cold fire-scar language into a readable visual beat. It is not a cave entrance gameplay marker, route script, nav/pathfinding feature, encounter lane, spawn marker, cave trigger, collision guarantee, DCC target, Unreal target, startup placement, final approval, implementation target, or Hermes task.

Blood Axe cave remnant dressing must stay separate from neutral/civilized Giant culture. The kit should read as hostile Giant raider residue: rough highland stone, soot, ash, mud, cave grit, rawhide, rope, restrained oxide red cloth, dull bone, sparse blackened iron, and cold abandonment. Do not use refined cave-town masonry, blue-gray civic stonework, warm hearth identity, terrace/waterwork forms, peaceful highland wayfinding, carved civic ornament, or restrained blue-rune culture as the default read.
